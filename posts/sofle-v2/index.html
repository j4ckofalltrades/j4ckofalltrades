PCB and Switches

PCB

The SofleKeyboard is a 6×4+5 keys column-staggered split keyboard with encoder support. Based on the Lily58, Corne, and Helix keyboards.

Switches

AEBoard Naevys v1 (Tactile, lubed Krytox 205g0)

  • Nylon bottom, PC top
  • Stem: MPE tactile stem - Custom xixed PE - A proprietary MPE blend that enables smoothness and good shrinking rates
  • 58g spring
  • 3 pin

Keycaps

Katha Baybayin

  • KAT profile
  • Smooth texture
  • 1.65mm thick PBT
  • Dye sublimated and reverse dye sublimated legends

Firmware

Modified version of default Via-compatible keymap with focus on adding a standard navigation cluster layer plus some Vim-inspired features e.g. soft escape (Esc when held, Ctrl when tapped), using h, j, k, l as arrow keys.

Layout

View in Keyboard Layout Editor

Features

  • Via support
  • Mode for soft escape (Esc when tapped, Ctrl when held)
  • Vim-style navigation (h j k l as arrow keys)
  • Mode for standard navigation cluster
  • Toggling between layers when encoders are pressed
  • Encoder controls (additional layer/s when Ctrl or Shift is held)
